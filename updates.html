11/15/2019 Project Update - Maid Chan 1.1.0 is released!
11/15/2019 项目更新 - Maid Chan 1.1.0 已经公开！

Maid chan 1.1.0 has been officially released! In this version, we included the following updates:
-Ability to react to basic greetings & commands(For example: @xxx hi)
-Ability to spam @Reaily#6672 (lol)
-Ability to assign roles to a member via command (Beta testing)
-Multi-language support between different servers (English and Chinese only for now)
-Move to a newer version of Discord API (discord.py rewrite)
-Ability to greet new members
-Music player(Beta testing)
 -We only support "play", "stop", "pause", and "resume" command for now
 We are planning to include the following in the future updates:
-A more stable music player
-Add 'loop', 'stop', 'skip', 'pause' , and 'queue' feature into the music player
-Limit the ability to assign roles via command to certain roles/users only
-Ability to conduct normal conversation using NPL models trained by Tensorflow
