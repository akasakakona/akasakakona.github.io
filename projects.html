Maid-chan
女仆酱


Project Maid-chan is inspired by the AI named "Maid-chan" from the anime The Pet Girl of Sakurasou. The ultimate goal of this project is to recreate her in real life. It is currently open source on GitHub so everyone is welcomed to make Maid-chan better!! Machine learning and more fun features will be implemented in the future!

女仆酱这个项目被动画《樱花庄的宠物女孩》中名为“女仆酱” 的AI所启发。这个项目的最终目的是在现实中重现动画中的女仆酱。目前这是一个在GitHub上开源的项目，欢迎大家随时改进女仆酱！在未来还计划加入机器学习等各种有趣的新功能！
